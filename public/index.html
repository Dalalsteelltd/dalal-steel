🔍 Visual Defect Reference Guide
🔍

Common Weld Defects — Visual Identification

🔴
Crack
Linear fracture in weld or HAZ. Zero tolerance — always reject.
🔴
Lack of Fusion
Weld metal not fused to base metal. Critical failure risk.
🔴
Incomplete Penetration
Root not fully fused. Fails all load-bearing requirements.
🟡
Porosity
Gas pockets trapped in weld. Acceptable only if < 3 mm, isolated.
🟡
Undercut
Groove along weld toe. Max 1 mm depth per AWS D1.1.
🟡
Overlap
Weld metal beyond toe line. Creates stress concentration.
🟢
Spatter
Weld metal droplets on base. Minor — grind if excessive.
🟢
Arc Strike
Mark outside weld zone. Grind smooth, re-inspect HAZ area.
🟢
Surface Oxidation
Heat discolouration. Wire brush and document finding.
